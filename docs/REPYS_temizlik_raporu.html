🛠️ Adım Adım Temizleme Rehberi

  1. Önce git commit alın! Temizlik öncesi mevcut hali kaydedin:
    git add -A && git commit -m "temizlik öncesi snapshot"
  2. Kritik sorunları manuel düzeltin: Boş f-string'leri (f"metin" → "metin"), üst üste tanımlı fonksiyonları ve ulaşılamaz kodları elle inceleyin. Bunlar araç çözemez.
  3. Kullanılmayan importları otomatik temizleyin:
    pip install autoflake
    autoflake --in-place --remove-all-unused-imports -r .
  4. Sonucu doğrulayın:
    python -m pyflakes . 2>&1 | wc -l
    Sayı ciddi ölçüde düşmeli.
  5. Kullanılmayan değişkenleri temizleyin: Her dosyayı ayrı açıp, atanan ama okunmayan değişkeni ya silin ya da _ ile adlandırın.
  6. Uygulamayı başlatıp test edin: python main.pyw çalışıyor mu? Her modülü kısaca gözden geçirin.
