# Menu Filtreleme ve Page Guard Akisi (Metinsel)

Tarih: 2026-02-25

Menu Filtreleme:
1. ayarlar.json okunur
2. Her menu item icin permission anahtari belirlenir
3. Kullanici yetkisi varsa item gosterilir
4. Yetkisi yoksa item gizlenir

Page Guard:
1. Menu item tiklanir
2. Page permission kontrol edilir
3. Yetki yoksa uyari gosterilir ve sayfa acilmaz
4. Yetki varsa sayfa olusturulur
